SMART BIN - USING GOOGLE'S TENSORFLOW TO IMPROVE RECYCLING RATES

Separating waste is sometimes confusing and cumbersome—something that can derail people's good intentions to recycle. Using TensorFlow, we built a “Smart Bin” that can identify types of trash and sort them automatically.

Used stack:

  • Google Tensorflow
  • Google Collab
  • Python
  • Arduino
https://www.youtube.com/watch?v=jtUbmNZLGVk

3D-Printed Myoelectric Prosthetic Arm

Built a 3D-printed robot hand that is controlled by muscle contractions captured as EMG signals from electrodes placed on the forearm. This can be used to develop low-cost prostheses that significantly improves an amputee's quality of life.

Used:

  • Servo motors for actuators for the bidirectional tendon system, Arduino Mega to control servo angle of rotation, and EMG sensors and electrodes to capture EMG signals.
https://www.youtube.com/watch?v=2EsYUN9ry9U

Pandemic Simulator

Built a web-based application that simulates the spread of a virus through a community and allows testing of measures like travel restrictions, social distancing, workplace/ school closures, hospital/ICU readiness on spread and fatality.

Used stack:

  • HTML5
  • CSS3
  • JavaScript
  • Google Charts
  • jQuery
http://vayunmathur.com/pandemic/index.php

Soong One: 3D-printed robot head with gaze control and facial recognition

Built 3D-printed robot head that uses a camera and a machine-learning neural network to recognize individuals and follow the gaze - a skill valuable for a more realistic interaction!

Used:

  • 6 servos which provide 6 degrees of motion, a custom printed circuit board that takes power and 4 PWM data channels and passes them along to the 4 servos in the head, the Adafruit I2C servo board that creates the servo PWM signals, Arduino Mega and Raspberry Pi 4 and Raspberry Camera v2.
https://www.youtube.com/watch?v=ffyatxbOjUA

Singapore Weather & Bus Station Alexa Skill

Built one of the first voice-bots in 2017 using the Singapore LTA API to check next arriving bus time. Featured by GovTech.

Used stack:

  • Alexa Service API
  • JavaScript
